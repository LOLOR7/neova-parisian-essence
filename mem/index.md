# Project Memory

## Core
NEOVA — Parisian luxury renovation studio. Site language: French.
Aesthetic: minimal, editorial, warm off-white bg, deep slate-blue typography, beige/stone accents. Generous white space.
Typography: Cormorant Garamond (display, light) + Inter (body, light). Never use playful or generic fonts.
No loud colors. No purple. No generic construction imagery. Calm, slow transitions.
Backend: Lovable Cloud. AI chat via edge function `chat` using Lovable AI Gateway (google/gemini-2.5-flash).
